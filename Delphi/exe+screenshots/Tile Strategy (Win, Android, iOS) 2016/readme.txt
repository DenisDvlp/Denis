Платформы: iOS, Android, Windows
Цель: пошаговая средневековая фентезийная стратегия
Особенности: поиск путей для персонажей через ячейки с разной проходимостью, генератор огромных карт, быстрый алгоритм подгрузки и отрисовки больших карт(10000x10000 тайлов одновременно), алгоритм объединения водных регионов, мультитачный зуминг(только для Android, iOS)
Рекомендации:
1. Откройте на весь экран окно
2. Нажмите Левую кнопку "Button1", она позволит отобразить карту
3. Нажмите Среднюю кнопку "Button1", она генерирует карту разных размеров и расставляет персонажей
4. Выделяйте ЛКМ разных персонажей, чтобы отобразить их доступные пути перемещения (само перемещение не работает)
5. Правая кнопка "Button1" показывает/скрывает лог
Примечание: разработка игры остановлена на ранней стадии по причине нехватки времени.
