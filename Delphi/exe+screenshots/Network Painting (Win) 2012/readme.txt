Платформы: Windows
Цель: рисование и общение по локальной сети
Особенности: соединение по протоколу UDP в локально сети, рисование и общение посредством чата одновременно с восьмью участниками
Рекомендации: 
1. запустите приложение на нескольких компьютерах в сети
2. раскройте список по кнопке под IP
3. введите вместо localhost IP того ПК, к которому желаете подключиться.
4. нажмите ON для соединения
5. рисуйте
6. раскройте чат правее и можете отправить сообщение (чекбоксы указывают кто из получателей получит сообщение)
Примечание: фича "очистить у себя" работает не правильно
